Revenge-Of-The-Data-Mod

A mod for Mindustry, including two new turrets. Update 1.2 is live! Introducing the Micro Manager and the Code Graph! The Micro Manager is the overpowered version of the Draug Miner drone. With a high-powered drill speed, along with a massive storage of 1000 materials, The Micro Manager will replace all copper and lead drills to save you space. The Code Graph is a minute version of the Revenant ship, with a fast velocity and a good firing rate. Warning: The .zip file may be broken. Please unzip the file and extract the original mod.
